A media-supported way to learn StatWiseAI

This prototype illustrates the basics of all nine tutorial modules using minimalist visuals and short interactive supports — decision trees, prompt builders, and checklists — instead of long blocks of text alone.

◆
Long-term goal, previewed here

In Phase 2, this same page shell hosts a chatbot sandbox where visitors can practice prompting a constrained model of StatWiseAI, alongside short videos that model real sessions. See the architecture plan for the phased roadmap.

How this path works

Module 1 is a gate: its rules apply to every later module. Five modules below are built out as working prototypes — Responsible Use, AI Basics for Researchers, Prompting for Data Analysis, Working with Dataset Documentation, and Reviewing AI Outputs — to demonstrate the media-support pattern. The remaining four are shown at the same visual fidelity as they will ship, marked Planned.
